The hostname:port configuration. Please specify a hostname (or IP address) and an optional port from which to run ZAP. For example, a value of localhost will launch ZAP on this server with the default port of 8090. Specify myhost:9876 or 10.7.7.7:9876 to bind ZAP to the port 9876 on the host with the hostname myhost or with the IP address of 10.7.7.7.
